# 1.不错先生

`不错先生`本名叫`史麦乐`，地道的中国东北程序员，写Java有20余年，150万行之多。
不要惊诧于那坨 `N🚫T B💩D`的组合头像，它并不属于`不屎先生`(Mr.NoShit)。
它是一个丧失了审美，还迷信自然数`654321`的家伙自己PS的。

你听过那个『不要！停！不要停！』的故事么？不错先生有过类似的经历，他的版本是这样的，

* 不！错！
* 不要错！
* 不错！
